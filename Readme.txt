No JavaScript, existem 6 tipos primitivos:
String.
Number.
Boolean.
Null.
undefined.
Symbol (novo no ECMAScript (en-US)

Variáveis. Você usa variáveis como nomes 
simbólicos para os valores em sua aplicação.
O nome das variáveis, chamados de identificadores,
obedecem determinadas regras. Um identificador 
JavaScript deve começar com uma letra, underline 
( _ ), ou cifrão ( $ ); os caracteres subsequentes
 podem também ser números (0-9).
